Node.js is one of the leading backend languages you can work with and can be used for building tools, desktop apps, mobile apps, databases, and more. Learn the foundations of Node.js so you can go forward and create fantastic JavaScript apps outside the browser!

## Topics
By the end of the workshop, you'll learn:

- Node.js Modules
- Internal Modules & npm
- ES Modules & Latest JS Features
- Asynchronous Patterns
- Debugging & Testing
- Publishing and Deploying custom modules
- and much more!

## Pre-requisites
- Attend my [JavaScript Fundamentals](/workshops/javascript-fundamentals) workshop, or have the equivalent basic understanding of JavaScript (variables, functions, objects, arrays, if statements)
- A computer with an Internet connection and Google Chrome installed
